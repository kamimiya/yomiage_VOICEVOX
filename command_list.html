yomiage_VOICEVOXコマンド一覧

2022/03/13時点で実装されているコマンド一覧です。

基本コマンド

!help
コマンドリストが表示されます。（簡略版）
!hello
バージョン情報を確認できます。
!join
BOTをボイスチャンネルに呼びます。（コマンド使用者が入っているボイスチャンネルに飛びます）
!leave
BOTをボイスチャンネルから退場させます。
!stop
再生している音声をストップします。

単語リストコマンド

!wlist add A B
読み仮名の登録をします。AをBと読ませます。
例）!wlist add gorilla ゴリラ
!wlist delete A
辞書登録の削除を行います。
例）!wlist delete gorilla
!wlist show
辞書登録の確認を行います。

調声コマンド

!chg_my_voice
ボイスを変えます。!chg_my_voice A Bで設定できます。A, Bは以下のように設定します。
A: zundamon, metan, tsumugi, ritsu, hau, takehiro, torataro, ryusei, himari, sora, mochiko
B: normal, amaama, sexy, tsun, whisper
例）!chg_my_voice metan normal
注意: tsumugi, ritsu,hau, takehiro, torataro, ryusei, himari, mochikoはnormalしか実装されていない。
!chg_speed
音声再生スピードを変えます。!chg_speed Aで設定できます。Aは0.5から2.0までの実数で指定してください。
チャンネル全体で変えます。また、Voibotを再起動すると設定がリセットされます。

各種設定

!inform_someone_come
入退室通知のオンオフを切り替える
!time_signal
時報のオンオフを切り替える
!read_name
名前読み上げ機能のオンオフを切り替える
!number_of_people
在室人数チェックのオンオフを切り替える
!auto_leave
自動退出のオンオフを切り替える
!word_count_limit
文字数制限の設定
例: 50文字に設定したい時）!word_count_limit 50
!show_setting
現在の設定を確認する
